This module is the main loop. It processes all the player, dungeon and mob objects and displays them in a meaningful way.

Libraries used:
os, random, dungeon, player, mob

Functions:

crawl():
This is the main function.
1.- Asks for a dungeon size (Returns message if smaller than 40x20)
2.- Creates a dungeon object, and then places a player object inside it, asking for a player name.
3.-Main loop
3.1.- If the player is in the exit tile, it creates another floor. This is done by creating a new dungeon, creating a new player and then assigning the old player variables to the new player using save(), load() and the launch.py temporary global variables.
3.2.- Fills the filled array (see dungeon) with the player data and displayr the minimap and basic stats and options
3.3.- Asks for an option, processes it and starts again.

lsave(player):
Takes the attributes from the player object and saves them in temporary global variables.

lload(player):
Takes the temporary global variables and replaces the attributes of the player object with them.

menu():
Main menu. Gives access to a new game and the game options. Exiting this menu will end the program.

newchar():
Now empty. The new character generator called from the main menu will go here.

options():
Displays and modifies the game options (Currently empty)

purge():
Resets the values of all the temporary global variables.
