Player class. Used to generate, define and manage the player.

Libraries used:
os, random, dungeon

Functions:

__init__(self, dungeon):
Class constructor. Initializes attributes, adds random race and class, and places the player at the coordinates of the entrance tile.

charsheet(self):
Brings up the character sheet menu. Displays name, attributes and optionsto save, load, modify the character and spend the experience gained.

getatr(self):
Prints the primary and secondary attributes in console.

invmenu():
Inventory managing menu (planned feature).

load(self):
takes attributes saved in a plain text file in ./save and loads the character with them.

move(self, dungeon, str direction):
Moves the player one tile in the given direction. Accepted strings are n, s, e and w.

optmenu():
Character options name.

save(self):
Takes the attributes from the player and stores them in a text file (./save).
The file has the format (attribute:value), one attribute per line.

secondary(self):
Calculates the secondary attributes from the player.

spend(self):
Brings up the experience spending menu.
